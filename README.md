## Setup

Follow these steps to set up the project locally:

### 1. Clone the Repository

```bash
https://github.com/Vaibhav-kesarwani/Face-detection
```

### 2. Navigate to the Project Directory

```bash
cd Face-detection
```

### 3. Install Dependencies

```bash
pip install -r
```

### 4. Run the App

```bash
python main.py
```
## Usage
